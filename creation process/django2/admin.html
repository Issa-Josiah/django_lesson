Django admin

checking admin page

step 1

To check admin page first of all confirm that in the projects urls.py the admin url is available

step 2

Then run the server using the command

python manage.py runserver

Step 3

Then open the browser and go to the url http://localhost:8000/admin

this will open an admin page and you will be forced to input a name and a password

Step 4

Input the superuser name and password you created earlier using the command

python manage.py createsuperuser

after inputting the name and password you will be redirected to the admin dashboard

Step 5

To input model go adn register the model in the admin.py and return to theadmin dashboard to view

Step 6

To arrange the things you want to see from the models add a string to the model

You can also change the view in the admin page

Step 7

You can add, delete and edit values or onjects from the admin page

To add value you add new Fist in the values

to delete you enter the object you want to delete and delete object

to edit you enter the object you want to edit and make changes
